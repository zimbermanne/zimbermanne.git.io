## 🚀 How to Run

1. **Download** all project files
2. Place `index.html` and `dashboard.html` in the same folder
3. Open **`index.html`** in your browser
4. Register or login (use any name)
5. Start posting!

**Note**: This is a **frontend-only** demo using `localStorage`. Data is saved in your browser.

---

## 🔧 How to Use

### Featured Posts
- Simulated M-Pesa payment flow
- Lasts **7 days** before auto-expiry

### Admin Access
- Login with name: **`admin`**
- Click **"👮 Admin"** button in dashboard
